# Wordle Game - CISC 3130 Project

This repository contains a Java implementation of the popular word guessing game **Wordle**, developed as part of a course project for **CISC 3130: Data Structures**.

## About

The game allows players to guess a hidden five-letter word within a limited number of attempts. After each guess, the game provides feedback indicating which letters are:
- Correct and in the correct position
- Present in the word but in the wrong position
- Not present in the word

This project demonstrates object-oriented programming principles and basic data structures used to handle game logic and user input.

## Features

- Console-based gameplay  
- Input validation and error handling  
- Color-coded or symbolic feedback for guesses  
- Organized in separate branches for different homework stages (`hw02`, `hw03`)
